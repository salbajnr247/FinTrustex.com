How to Get Your Binance API Keys

  1. Log in to your Binance account.
  2. Navigate to "API Management" in your account settings.
  3. Click "Create API" and complete any security verification required.
  4. For security reasons, enable the following permissions only:
    • Read-Only (if you only want to track your portfolio)
    • Enable Trading (if you want to execute trades)
    • Do NOT enable withdrawal permissions
  5. Copy your API Key and Secret Key and enter them below.
  6. For added security, restrict API access to your IP address in the Binance settings.

Security Recommendations

  • Create API keys that are specific to FinTrustEX.
  • Restrict API access to your IP address when possible.
  • Regularly review and audit your API keys on Binance.
  • Enable two-factor authentication on your Binance account.
  • Use a strong, unique password for your Binance account.
  • Never enable withdrawal permissions for API keys used with third-party services.
