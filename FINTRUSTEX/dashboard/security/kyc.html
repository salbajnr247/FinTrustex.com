Verification Requirements

Accepted ID Documents

  • Valid passport
  • Government-issued ID card
  • Driver's license

Accepted Address Proof

  • Utility bill (issued within the last 3 months)
  • Bank statement (issued within the last 3 months)
  • Tax document (issued within the last year)
  • Government-issued letter (issued within the last 3 months)

Document Requirements

  • Documents must be valid and not expired
  • Full name must be clearly visible
  • Photo must be clearly visible (for ID documents)
  • Documents must be in color (not black and white)
  • File size not exceeding 5MB per file
  • Accepted formats: JPG, PNG, PDF
  • Documents must not be digitally modified or tampered

Verification typically takes 24-48 hours after submission. You will be notified via email once your verification is complete.
