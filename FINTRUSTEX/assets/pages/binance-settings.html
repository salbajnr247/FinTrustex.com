Security Information

  • Create an API key with only trading permissions (not withdrawal permissions)
  • Restrict API access to specific IP addresses when possible
  • Enable two-factor authentication (2FA) on your Binance account
  • Review your API key permissions regularly

How to Create a Binance API Key

  1. Log in to your Binance account

    Visit Binance.com and log in to your account

  2. Navigate to API Management

    Go to your account profile and select "API Management"

  3. Create a new API key

    Click "Create API" and follow the security verification steps

  4. Set appropriate permissions

    Enable "Enable Reading" and "Enable Spot & Margin Trading" only

    Do NOT enable withdrawal permissions for security reasons

  5. Set restrictions (recommended)

    Restrict access to specific IP addresses when possible

  6. Copy your API Key and Secret

    Save your API Key and Secret in a secure location

    Your API Secret will only be shown once
